# Exercício de Manipulação com jQuery - EBAC

Este projeto demonstra a aplicação da biblioteca jQuery para facilitar a manipulação de elementos HTML e a criação de interatividade dinâmica.

## O que foi desenvolvido:
* **Seletores Dinâmicos:** Uso da sintaxe jQuery para captura e manipulação de elementos do DOM.
* **Efeitos e Animações:** Implementação de transições visuais (fade, slide) para melhorar a experiência do usuário.
* **Gestão de Eventos:** Criação de funções para tratamento de cliques e interações com formulários.
* **Manipulação de Listas:** Lógica para adição dinâmica de elementos em listas e controle de estados (ex: marcar como concluído).

## Tecnologias:
* HTML5 / CSS3
* JavaScript
* jQuery
* Git / GitHub
